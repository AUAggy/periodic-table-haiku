# Aggy's Periodic Table: Haiku Edition

Nature is not as simple as dry textbooks suggest. It is a series of accidents and energetic personalities. 

This project is a single-page HTML application featuring 118 interactive elements. Every description is rewritten as a haiku: 5-7-5 syllables of humor, sass, or dark truth.

[![Aggy's Periodic Table](https://raw.githubusercontent.com/AUAggy/periodic-table-haiku/main/screenshot.png)](https://AUAggy.github.io/periodic-table-haiku/)

### Sample Poetry

*   **Beryllium**: Strong and lightweight gem. Do not breathe my toxic dust. Lungs will surely fail.
*   **Arsenic**: Shadow in the glass of wine. Lethal legacy.
*   **Astatine**: Blink and it is already gone. Ghost of the table.

### Technical Overview

The application is a self-contained HTML file. It uses CSS Grid for the table layout and JavaScript for interactivity. No external libraries or tracking scripts are included. Scientific data: including atomic mass and electron configurations: remains accurate to standard references.

### Usage

Open `index.html` in a modern web browser. Click any element to view its scientific data and corresponding haiku.
